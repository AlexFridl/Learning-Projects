# Responsive Bootstrap Webpage

This project demonstrates how to create a **responsive webpage** using **Bootstrap 5.3.3**.  
The layout is structured with Bootstrap’s **grid system** (`row` and `col-` classes) to ensure that the page adjusts automatically across different screen sizes — from desktop to mobile devices.

---

## Technologies Used

- **HTML5** – for page structure  
- **CSS3** – for custom styling  
- **Bootstrap 5.3.3** – for responsive grid layout and components 

---

## Responsive Design Overview

- The layout is built using Bootstrap’s **container**, **row**, and **col** classes.  
- Elements automatically stack or align side-by-side depending on the screen width.  
- The design is optimized for desktop, tablet, and mobile devices.  
- Example classes used:
`col-2`, `col-8`, `col-12`, 
`col-sm-3`, `col-sm-4`,`col-sm-6`, `col-sm-8`, `col-sm-12`,
`col-md-4`,`col-md-6`, 
`col-lg-2`, `col-lg-3`, `col-lg-4`, `col-lg-6`, `col-lg-7`, `col-lg-8`, `col-lg-9`, `col-lg-10`,    `col-lg-12`.

## Struktura projekta


## Struktura i responzivnost

- Koristi se **Bootstrap grid sistem** (`container`, `row`, `col-`) za raspored elemenata.  
- Elementi se automatski reorganizuju u više redova na manjim ekranima.  
- Korišćene klase: `col-12`, `col-md-6`, `col-lg-4` za prilagođavanje layout-a.  
- Testirano na različitim veličinama ekrana (desktop, tablet, mobilni).
